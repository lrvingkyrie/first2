一.图像和连接
  1.URL
    1.绝对路径
    2.相对路径
    3.根相对路径
      从web站点的根目录开始去查找，以/开始
      ex:/codeboy/img/logo.png
  2.图像
    1.图像格式
      .jpg 压缩比例较大，比较模糊
      .png 背景透明
      .gif 动态图
    2.语法
      标记：<img>
      属性：
         src: source 源，设置图像路径
	 alt：图像出错时，所给的提示文字
	 width:设置图片的宽度，取值以px或%为单位的数值
	 height:设置图片的高度，取值以px或%为单位的数值
      练习：
         在网页显示三张图片，要求分别以相对路径中的三种形式来引入图片，然后设置图片的宽高以及提示文字。

  3.链接
    1.语法
      标记：<a></a>
      属性：
         1.href:链接路径
	 2.target:指定打开链接的方式
	   取值：
	     1._self 默认值，在当前标签页中打开新网页
	     2._blank 在新的标签页中打开网页
    2.其它的表现形式
      1.下载资源
        让链接的url，链接到rar/zip文件
	<a href="*.zip/rar">下载</a>
      2.返回当前页面顶部
        <a href="#">返回页面顶部</a>
      3.链接到电子邮件
        <a href="mailto:g-yangyan@tedu.cn">新建邮件</a>
      4.链接到javascript，执行脚本
        <a href="javascript:js代码">执行js代码</a>
    3.锚点
      1.什么是锚点
        锚点，就是网页中的一个记号，可以通过超链接的方式链接到记号位置处。
      2.锚点的使用方式
        1.定义锚点
	  1.使用a标记的name属性定义锚点
	    ex:<a name="锚点名称"></a>
          2.使用任意标记的id属性定义锚点
	    ex:<any id="锚点名称"></any>
	    
	2.链接到锚点
	  1.链接到当前页的锚点
	    <a href="#锚点名称"></a>
          2.链接到其它网页的锚点
	    <a href="网页的url#锚点名称"></a>
        练习：
	  1.新建网页02-exercise.html，
	  在网页中分别用div和h2定义两个锚点，然后通过a标记链接到以上锚点位置。再在其它网页中用p元素定义一个锚点，锚点名称随意，通过02-exercise.html中的a标记链接到p元素这个锚点。

二.表格
  1.表格语法
    1.标记
      表格：<table></table>
      行：  <tr></tr>         ----table row
      列/单元格：<td></td>    ----table data
      练习：
        创建一个4行4列的表格
	  1.在网页中创建一个<table></table>标记
	  2.在<table>标记中嵌套<tr></tr>
	  3.在<tr>中嵌套<td></td>
    2.属性
      1.table
        border:设置表格的边框 
	width:设置表格的宽度
	height:设置表格的高度
	align:设置表格的水平对齐方式
	      取值：left/center/right
        bgcolor:设置背景颜色
	cellpadding:设置单元格内边距(内容与边框之间的间距)
	cellspacing:设置单元格外边距(边框与边框之间的间距)
	练习：
	  设置表格的尺寸为400*400px,
	  设置表格的背景颜色为pink
	  设置表格的边框为1px
	  设置表格水平居中显示
	  设置表格的内边距为5px
	  设置表格的外边距为10px
      2.tr
        1.align 设置当前行内容的水平对齐方式
	2.valign 设置当前行内容的垂直对齐方式
	  取值：top/middle/bottom
	3.bgcolor 设置当前行的背景颜色
      3.td
        1.width 宽度
	2.height 高度
	3.align 水平对齐
	4.valign 垂直对齐
	5.bgcolor 背景颜色
	6.colspan 跨列
	7.rowspan 跨行
    3.不规则表格的使用
      1.跨列
        属性：colspan
	从指定单元格的位置处开始，横向向右合并几个单元格（包含自己），被合并掉的单元格要删除。
      2.跨行
        属性：rowspan
	从指定单元格的位置处开始，纵向向下合并几个单元格（包含自己），被合并掉的单元格要删除。
    4.可选标记
      1.表格标题
        标记：<caption></caption>
	如果要设置标题，必须将<caption>标记放在<table>之后
      2.行/列标题
        标题：<th></th>
    5.表格的复杂应用
      1.行分组
        可以将连续几个行，划分到一个组中，进行统一管理。
	1.表头行
	  <thead></thead>
	  表格中最上面一行放在thead中
        2.表主体
	  <tbody></tbody>
	  允许将若干行放在tbody中进行统一管理
	3.表尾行
	  <tfoot></tfoot>
	  表格中最后一行放在tfoot中
      2.表格的嵌套
        表格中所有的嵌套都只能放在td中
三.列表（重点）
  1.列表的作用
    按照从上到下（或从左到右）来显示所有的数据，并且可以在数据前增加标识。
  2.列表的组成
    列表都是由"列表类型"和"列表项"来组成。
    1.列表类型
      有序列表：<ol></ol> order list
      无序列表：<ul></ul> unorder list
    2.列表项
      用于表示列表中的数据（嵌套在列表类型中）
      列表项：<li></li> list item
  3.列表属性
    1.有序列表
      1.type 作用：指定标识项的类型
        取值：
	   1：默认值
	   a：小写字母
	   A：大写字母
	   I：大写罗马数字
	   i：小写罗马数字
      2.start 作用：指定起始编号的位置 （从第几个位置开始排序）
        取值：无单位的数字
    2.无序列表
      1.type 作用：指定列表标识类型
        取值：
	   disc 实心圆
	   circle 空心圆
	   square 实心方块
	   none 不显示标识
  4.列表嵌套
    在一个列表中去嵌套另一个列表
    被嵌套的列表只能出现在li中
    ex:
    <ul>
      <li>
         数据
        <ol>
	  <li>数据</li>
	</ol>
      </li>
    </ul>
  5.定义列表
    1.什么是定义列表
      定义列表常用于给出一类事物或对名词的解释说明等。
    2.语法
      1.<dl></dl> 表示定义列表
      2.<dt></dt> 表示定义列表中要解释说明的名词
      3.<dd></dd> 表示定义列表中对名词的解释内容
       <dl>
         <dt></dt>
	 <dd></dd>
       </dl>
      使用场合：图文混排。
